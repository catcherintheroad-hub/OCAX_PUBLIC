# OCAX - 让闲置设备变成算力

---

## OCAX 是什么？

**一个算力交易平台，连接有闲置设备的人和需要算力的人。**

你有一台闲置的手机、平板或电脑？
接入 OCAX。你不用的时候，它帮别人跑真实任务。
你赚钱，他们完成工作。

你需要更多设备来处理工作？
在 OCAX 上发布任务，花钱让别人帮你跑。

---

## 跟其他项目的区别

| 那些项目 | OCAX |
|----------|------|
| 假任务、空气盘 | 真实任务、真实用户 |
| 概念口号 | OpenClaw 实际集成 |
| 躺着赚钱 | 任务完成才能拿钱 |
| 不透明 | 开源、公开仓库 |

**你的设备跑的是真实任务。你赚到的是实际收益。**

---

## 设备闲着的人

**你的设备其实能做更多。**

大部分设备利用率只有 5-10%，剩下的都浪费了。

OCAX 让你把这部分时间变现。

- 把设备接进来
- 空闲时跑真实任务
- 任务完成，你拿钱

跟租房子一样，不过是租算力。

---

## 需要更多设备的人

**不想买更多硬件？可以租。**

- 项目需要更多设备？
- 想让任务跑在多台机器上？
- 一次性任务硬件不够？

OCAX 让你从已连接的设备调用算力。

用多少付多少，可大可小。

## 怎么开始

**有设备的人：**
1. 安装 OpenClaw
2. 接入 OCAX 网络
3. 让设备开始工作

**发布任务的人：**
1. 发布你的任务
2. 设备帮你完成
3. 任务完成，付款
