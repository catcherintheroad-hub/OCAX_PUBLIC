# OCAX MVP Page Specs - Public Overview v1

**Version**: 1.0  
**Type**: Public Overview  
**Date**: 2026-03-18

---

## Overview

OCAX MVP pages are designed for simplicity. Each page has a clear purpose and shows only what's needed.

---

## Pages

### For Node Providers

**Node Dashboard**
- Status toggle (online/offline)
- Reputation (tier, completion rate)
- Earnings (today, total)
- Current task

**Task Assignment**
- Current task details
- Accept/reject buttons
- Checkpoint progress

### For Task Requesters

**Task Dashboard**
- All tasks overview
- Status badges
- Price and progress

**Create Task**
- Task form
- Price calculator
- Submit button

**Task Detail**
- Task status
- Checkpoint progress
- Settlement info

### For Admins

**Admin Dashboard**
- System stats
- Recent activity
- Alerts

**Whitelist Queue**
- Pending applications
- Approve/reject buttons

**Node Management**
- All nodes list
- Status and reputation

---

## Key Data Shown

### Node Providers See
