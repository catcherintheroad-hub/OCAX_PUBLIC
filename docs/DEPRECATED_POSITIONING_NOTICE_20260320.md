# Deprecated Public Messaging Notice

Date: 2026-03-20

Some older files in `docs/` and `release/` reflect an earlier OCAX public narrative centered on:

- idle devices
- passive income
- computing marketplace language
- broad public user acquisition

These materials no longer represent the current OCAX product positioning.

> OCAX is a verified OpenClaw execution network for running tasks on real computers.

Chinese:

> OCAX 是一个调用真实电脑执行 OpenClaw 任务的可验证执行网络。

If an older file conflicts with this positioning, treat the older file as deprecated historical material.
