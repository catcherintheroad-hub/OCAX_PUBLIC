# OCAX 平台推广计划

## 平台简介

**OCAX** - 去中心化计算资源交易平台

让计算资源提供者通过共享空闲算力获得收益，让需求者以更低成本获取计算资源。

## 核心功能

| 模块 | 功能 |
|------|------|
| OCAX-Passport | 节点身份证 + 硬件评分 |
| OCAX-Validator | 自动验收任务 |
| OCAX-Settler | 自动结算 + 多支付方式 |
| OCAX-Exchange | 任务市场 + 智能匹配 |

## 优势

1. **去中心化** - 点对点交易，无中间商抽成
2. **自动化** - 验收/结算全流程自动
3. **多样化支付** - Stripe/PayPal/Crypto/支付宝/微信
4. **灵活结算** - 即时/延迟/分期
