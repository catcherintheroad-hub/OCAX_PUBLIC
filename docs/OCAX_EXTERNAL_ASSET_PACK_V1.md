## 1. 项目定位摘要

### 1.1 一句话定位

> OCAX - 去中心化计算资源积分市场

### 1.2 三句话简介

1. **OCAX** 是一个基于积分的去中心化计算资源交易市场。
2. 节点贡献 GPU/CPU 算力，获得 Credits (CR) 和 Trust Points (TP) 积分奖励。
3. 积分可随平台升级平滑过渡到真实交易。

## 4. 积分机制摘要

### 4.1 TP (Trust Points) - 声誉积分

| 来源 | 奖励 |
|------|------|
| 激活节点 | +30 TP |
| 验收通过 | +5 TP |
| 满分 | +10 TP |
| 连续通过 | +15 TP |
| 稳定性奖励 | +30 TP/月 |

**用途**: 等级、匹配权重、信誉

### 4.2 CR (Credits) - 经济积分

| 来源 | 奖励 |
|------|------|
| 验收通过 | 基础 CR |
| 满分 (+30%) | 基础 CR × 1.3 |

**用途**: 任务结算、未来真钱映射

### 4.3 为何用积分替代真钱

| 原因 | 说明 |
|------|------|
| 降低门槛 | 无需真实资金即可参与 |
| 风险可控 | 积分无金融属性 |
| 平滑升级 | 满足条件后可升级到真钱 |
| 合规友好 | 规避监管风险 |
