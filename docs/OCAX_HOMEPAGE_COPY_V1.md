## What OCAX Is

### Short Copy

OCAX routes tasks onto real computers running OpenClaw.

It is designed for workflows that need:

- real desktop environments
- real browser environments
- local machine context
- validation before settlement

### Expanded Copy

OCAX is not a generic task board and not a broad open compute marketplace.
It is a controlled execution network for tasks that do not fit neatly into generic cloud or VPS setups.

## How It Works

### 4-Step Version

1. Caller submits a task
2. An approved node runs it on a real computer with OpenClaw
3. The platform validates the output
4. Accepted work moves into settlement and records

### Short Loop Line

Submit -> Execute -> Validate -> Settle
