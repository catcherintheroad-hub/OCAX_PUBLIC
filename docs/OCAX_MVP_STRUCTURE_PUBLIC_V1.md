## What Is OCAX MVP?

OCAX MVP is a **task collaboration network** where:
- Device owners contribute their idle hardware
- Task requesters get work done
- Everyone gets paid for real work

---

## User Roles

| Role | What They Do |
|------|-------------|
| Node Provider | Offers idle device for tasks |
| Task Requester | Posts tasks that need compute |
| Platform Admin | Manages whitelist and quality |

---

---

## How It Flows

### Node Joining

1. Apply for whitelist
2. Admin reviews
3. Get approved
4. Start receiving tasks

### Task Lifecycle

1. Create task
2. Task assigned to node
3. Node completes checkpoints
4. Task finishes
5. Payment released

---

## MVP Success

**4 indicators must ALL be met:**

| Indicator | Target |
|----------|--------|
| Tasks completed | ≥ 10 |
| Checkpoint success | ≥ 80% |
| Node retention | ≥ 70% |
| Security incidents | 0 |

---

## What's NOT in MVP

- Open marketplace
- API access
- Mobile app
- Complex analytics
- Task bidding
