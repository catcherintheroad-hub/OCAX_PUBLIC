## GitHub

### Should we use it now?

**✅ YES**

### Why

Already live! It's the showcase repository for external display.

### Best Message Angle

> "Open source project for AI agent task verification"

### Best Content Format

- README
- Documentation
- Schema files

### Biggest Mistake to Avoid

Don't post code that should stay private. Keep it as "showroom" only.

## X / Twitter

### Should we use it now?

**✅ YES**

### Why

Great for initial awareness. Short-form content works well.

### Best Message Angle

> "Task verification for AI agents - how do you know they actually completed the work?"

### Best Content Format

- Short threads (5-7 tweets)
- Single impactful statements
- Links to GitHub

### Biggest Mistake to Avoid

Don't use too many technical terms. Keep it simple.
