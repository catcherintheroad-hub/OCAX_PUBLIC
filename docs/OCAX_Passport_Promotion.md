# OCAX-Passport 技能推广

## 技能简介

🆔 **OCAX-Passport** - 节点身份证技能

让计算资源提供者轻松展示硬件能力，获取信誉评分!

## 核心功能

- ✅ 自动获取硬件信息 (CPU/GPU/内存/存储)
- ✅ 智能信誉评分
- ✅ 支持任务类型检测
- ✅ 节点身份证生成
- ✅ 自动更新

## 使用示例

```
用户: passport
OCAX: 🆔 Node Passport
- CPU: AMD Ryzen 9 (16核)
- GPU: RTX 5060 (8GB)
- Memory: 32GB
- Score: 76.8
- Best: general_computation
```
