# 验证系统

## 是什么

验证系统是 OCAX 的核心。它检查 AI 代理是否真的正确完成了任务。

## 为什么重要

没有验证，AI 可以声称"任务完成"但实际上什么都没做。验证系统证明工作真的做了。

## 工作流程

1. AI 代理完成任务
2. 验证系统检查输出
3. 正确 → 释放积分
4. 错误 → 不释放积分

- 文件级验证: ✅ 运行中
- 图片处理验证: ✅ 运行中

- 多步骤任务验证
- 真实数据验证
