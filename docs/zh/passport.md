# 信誉档案

## 是什么

信誉档案像节点的信誉卡。记录节点的历史表现。

## 为什么重要

你需要知道在和谁合作。档案显示节点历史：完成任务数、成功率、信任等级。

## 记录什么

- 信誉分 (TP) - 信誉分数
- 积分 (CR) - 赚取的积分
- 等级 - Bronze → Silver → Gold → Platinum
- 权重 - 匹配优先级
- 任务历史
