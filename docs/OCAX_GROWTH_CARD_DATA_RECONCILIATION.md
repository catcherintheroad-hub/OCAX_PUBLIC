# OCAX Growth Card 数据口径校正报告

**日期**: 2026-03-17  
**校正版本**: v0.3

---

## 1. 数据偏差分析

### 1.1 旧数据 (Growth Card 显示)

| 字段 | 旧值 | 来源 |
|------|------|------|
| TP | 140 | v0.2 规则计算 |
| CR | 190 | v0.2 规则计算 |
| Level | L2 (Silver) | TP=140 时 |
| Weight | 1.2x | L2 对应 |

### 1.2 新数据 (v0.3 规则)

| 字段 | 新值 | 说明 |
|------|------|------|
| TP | 75 | v0.3: 激活30+满分20+首成10+连击15 |
| CR | 117 | v0.3: resize(65) + convert(52) |
| Level | L1 (Bronze) | TP=75 时 |
| Weight | 1.0x | L1 对应 |

---

## 2. 偏差原因

| 原因 | 说明 |
|------|------|
| **规则版本不同** | Growth Card 读取的是 v0.2 计算的数据 |
| **满分 bonus 收紧** | v0.3 满分 bonus 从 50 改为 Base×30% |
| **TP 来源收紧** | v0.3 TP 来源更严格，减少了奖励 |
| **数据未同步** | Passport economy 字段未随规则更新刷新 |

---

| image_format_convert | 90 | 52 | -38 |
| **总计** | **190** | **117** | **-73** |

## 6. 校正结论

- **原因**: 规则版本不一致导致数据偏差
- **解决**: 以 v0.3 为准重新计算
- **结果**: Level 降为 L1 (Bronze)，CR 调整为 117
- **影响**: 更加合理的风控，防止奖励膨胀
