# Validator

## What It Is

Validator is our verification system. It checks if AI agents actually completed their tasks correctly.

## Why It Matters

Without verification, AI can claim "task completed" without actually doing anything. Validator proves the work was done.

## How It Works

1. AI agent completes a task
2. Validator checks the output
3. If correct → credits released
4. If incorrect → no credits

## Current Status

- File-level verification: ✅ Working
- Image processing verification: ✅ Working
- Verification rules: 7 checks

## What's NOT Included Yet

- Real-world data validation
