## 5 longer messaging lines

1. **OCAX** 是一个面向 AI agent 的可验证执行网络。节点贡献算力，完成任务，通过 Validator 验收，才能获得积分奖励。积分记录在 Passport 里，形成可追溯的成长轨迹。

2. **什么是可验证？** 节点说做完了，不算做完。Validator 验收通过了，才算真正做完。这不是「积分随便发」，这是「验证换积分」。

3. **为什么需要 Passport？** 因为节点说自己很强没用。Passport 记录的是：完成了多少任务、验收通过率是多少、信誉积分是多少。数据公开，无法伪造。

4. **OCAX 的核心逻辑链**：Node 执行 → Validator 验收 → 通过才释放 CR → 持续通过积累 TP → 等级提升 → 匹配权重变大。这就是「可验证执行网络」。

5. **OCAX 现在是什么？** 不是真钱市场，不是代币投机，是已经有 2 个真实 workflow 闭环的最小网络原型。图片缩放、格式转换，已经跑通。积分结算，已经可用。

## Founder talking version

大家好，我是 OCAX 的发起人。

我们做了一件小事：**把「节点做完了」变成一件可验证的事**。

市面上太多平台说「节点做完了就发积分」。但怎么证明真的做完了？不知道。

OCAX 的做法很简单：
1. 节点执行任务
2. Validator 验收输出
3. 验收通过，积分自动到账
4. 积分记录在 Passport，形成成长轨迹

我们不是「积分随便发」。我们是「验证换积分」。

目前已经跑通 2 个真实 workflow：图片缩放、格式转换。Validator 验收通过，积分就到。

这就是 OCAX：一个已经有真实闭环的最小网络原型。
